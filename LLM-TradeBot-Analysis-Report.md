   - Forms adversarial verification chain

   - DeepSeek, OpenAI (GPT), Claude, Qwen, Gemini
   - Seamless switching via Dashboard

   - Manage multiple exchange accounts
   - Unified API abstraction (Binance supported)

   - Multi-timeframe data (5m/15m/1h) fetched concurrently
   - Ensures data alignment at snapshot moment

6. **Safety First**
   - Stop-loss direction correction
   - Capital pre-rehearsal
   - Veto mechanisms for live trading

7. **Full-Link Auditing**
   - Every decision's adversarial process recorded
   - True "White-Box" decision-making

8. **Professional Backtesting**
   - Strategy validation before live trading
   - Access at http://localhost:8000/backtest
